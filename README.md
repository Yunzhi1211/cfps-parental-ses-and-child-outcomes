# CFPS Parental SES and Child Outcomes

This repository contains code and supporting materials for a research project on parental socioeconomic status (SES) and child outcomes based on CFPS-related data.

## Current status

This repository is still being organized and updated.  
The project materials are not yet final.

- Poster: currently under revision
- Draft paper: currently under revision

Some scripts, file names, and folder structures may continue to change as the project develops.

## Repository contents

### `1_do/`
Stata do-files used for data cleaning, sample construction, regression analysis, robustness checks, mechanism analysis, and graph production.

### `R Code for Poster Figure and Other/`
R scripts and related materials used for poster figures, visualization, and supplementary exploratory analysis.

## Data

Raw data are not included in this repository.

This is because:

1. some files are too large for GitHub;
2. the underlying data may be subject to access restrictions;
3. this repository is intended to share code and project structure rather than redistribute raw datasets.

Researchers who wish to replicate the analysis should obtain the relevant data through the proper official channels and configure local file paths accordingly.

## Reproducibility note

To run the code successfully, users may need to:

- obtain access to the relevant CFPS datasets;
- place raw data in a local data folder not tracked by Git;
- update file paths in the scripts based on their own machine environment.
