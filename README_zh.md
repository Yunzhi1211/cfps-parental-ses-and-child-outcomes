
# 中国代际流动性研究：基于 CFPS 面板数据的父母社会经济地位与子女职业成就分析

---

## 项目概述

本仓库包含了一项关于中国**代际社会经济流动性**纵向研究的完整分析代码与文档。利用 2010–2022 年中国家庭追踪调查（CFPS）数据，我们探讨了父母社会经济资源（SES）如何影响子女的职业成就，并特别关注了时间趋势、地理差异以及不同人群子样本的异质性影响。

---

## 研究重点
- 量化跨越七轮调查的代际 SES 传递弹性
- 识别父母 SES 发挥作用的机制（职业、教育、政治资本）
- 评估流动过程中的地理不平等
- 测试不同职业分类系统和模型设定下的稳健性

---

## 数据处理亮点与技术创新

### 1. **智能多波次面板构建与严格验证**

- 系统整合了 **7 轮 CFPS 调查数据**（2010, 2012, 2014, 2016, 2018, 2020, 2022）
- **自动负值编码转换**：实施了全面逻辑，将所有负值转换为缺失值（排除 ID 和时间变量）
- **波次-年份验证**：自动生成波次和年份标识符并进行验证检查
- **唯一性强制执行**：应用 `isid pid year` 验证以确保面板数据的唯一性；如果检测到冲突，脚本将中止并输出诊断结果
- **智能缺失值诊断**：通过注释代码记录了过去的数据发现（例如，父亲/母亲收入在 2016 年后缺失率 >90%，因此改用缺失率约为 10% 的家庭人均收入 `family_income`）
- **输出压缩**：自动应用 Stata `compress` 命令，在保持精度的同时减小文件体积

---

### 2. **基于分年度 PCA 的多维 SES 指数**

- 构建了结合 **3 个核心领域** 的综合父母 SES 指数：
  - **职业声望**：父亲和母亲的国际社会经济地位指数 (ISEI)
  - **人力资本**：父亲和母亲的受教育年限
  - **政治资本**：父母任何一方的中共党员身份标识
  
- **分年度而非混合 PCA**：为每个调查年份运行独立的主成分分析模型
  - *原理*：捕捉这些维度权重随时间演变的趋势
  - *实现*：对 `levelsof year` 进行循环，并设置每年度样本量阈值（N≥20）
  - *评分*：将分年度得分保存至面板变量中用于回归分析

---

### 3. **地理分层与抽样偏差校正**

- **省级汇总**：计算子女居住省份的均值和样本量
- **样本量过滤**：排除样本量 N<50 的省份，以确保统计稳定性
- **基于代表性的排除**：明确排除了西藏和新疆，原因如下：
  - CFPS 在这些地区的城市抽样偏差严重（牧民/农民覆盖不足）
  - 样本量过小导致误导性的地理模式
  - *文档说明*：如果不排除，由于抽样偏差，西藏会因样本结构问题显示为“深蓝色”（最高 ISEI 均值）
- **几何匹配**：创建 HASC_1 代码（分级行政区划代码），将 31 个省份与地图形状文件关联，用于绘制分级统计图
- **统一色阶**：所有 7 年的地图均使用基于汇总省级均值计算的全局 6 分位数断点

---

### 4. **全面的多维度稳健性框架**

**模型设定测试：**

| 维度 | 变体 | 目的 |
|-----------|----------|---------|
| **固定效应** | 仅省份；省份 + 年份 | 控制时间混杂因素 |
| **异常值处理** | 原始值；缩尾处理 (1%-99%) | 对极端值的敏感性测试 |
| **子样本** | 城镇/农村；男性/女性 | 异质性影响分析 |
| **结果变量** | ISEI; SIOPS; EGP 类别 | 测量指标的稳健性 |
| **控制变量** | 基础模型；增加家庭收入对数 | 路径识别设定 |

- **统一变量命名**：模型通过交换 `y_core` 和 `x_core` 的定义，实现用单一汇总表对比所有设定
- **系数提取流水线**：使用 `postfile` 记录分年度估计值及其标准误，以便进行趋势可视化
- **聚类标准误**：全程使用省份层面的聚类，以处理地理相关性

**输出成果：**
- 独立的稳健性表格（6_1 至 6_7）
- 单一精炼的汇总表（6_8），包含明确的参数矩阵
- 展示效应量一致性的系数图（Coefficient plots）

---

### 5. **基于 R 语言弦图的“观测-预期”流动分析**

**可视化：**

R 语言弦图实现计算了独立模型下的 **观测/预期 (O/E) 比率**：

```
预期值(ij) = (行总计_i × 列总计_j) / 总样本量_n
O/E 比率 = 观测值(ij) / 预期值(ij)
```

**判读规则：**
- **红色连线** (O/E ≥ 1.25)：代表过度出现的来源-目的地流动（流动溢价）
- **蓝色连线** (O/E ≤ 0.75)：代表出现频率低于预期的流动（流动壁垒）
- **浅灰色连线** (0.75 < O/E < 1.25)：接近独立模型的预期

**技术特点：**
- **基于三分位数的父母 SES 分组**：将 PCA 得分分为 低/中/高 三组
- **EGP 映射至连续得分**：将 9 类 EGP 阶层映射为类 SIOPS 得分用于 PCA 指数构建
- **行向父母聚合**：取父母双方受教育程度、职业声望和党员身份的最大值
- **多轨道圆形布局**：外圈标注类别；内圈显示计数和罗马数字
- **定向渲染**：通过间距设置在来源域和目的地域之间创造视觉分隔

---

### 6. **带系数追踪的分年度回归**

**生成的趋势可视化：**
- **散点图 + 置信区间 + 趋势线**：分年度的点估计值及其 95% 置信区间
- **叠加拟合线**：七条回归线使用年度特定颜色，展示 SES 弹性的演变
- **分面面板图**：2×4 矩阵显示所有年份的散点图及拟合线

---

### 7. **使用 R 语言漏斗图记录样本流向**

R 脚本通过以下方式可视化样本损耗：
- **横向条形图**：每一步显示样本量及相对于初始样本的百分比
- **自动换行标签**：长文本在 22 个字符处自动换行（响应式设计）
- **蓝色渐变配色**：从浅到深的视觉层级
- **右侧标签**：在条形外侧直接显示计数和百分比

## 核心方法论特点

### 时间设计
- **面板结构**：包含 pid-year 观测值的非平衡面板（2010–2022）
- **分年度 PCA**：每波次的父母 SES 指数捕捉了特定时期的测量结构
- **趋势提取**：通过分年度系数实现对流动弹性演变的直观检查
- **叠加回归**：七条拟合线展示了 SES 传递的一致性或长期变化

### 职业分类
采用多种结果变量确保结论不因单一分类法而产生偏差：
- **ISEI** (国际社会经济地位指数)：连续标尺，范围 16–90
- **SIOPS** (标准国际职业声望量表)：另一种职业声望度量
- **EGP** (Erikson-Goldthorpe-Portocarero)：用于阶层流动分析的九分类系统

### 抽样策略与代表性
- **省级变异**：涵盖 31 个省份（因记录原因排除西藏、新疆）
- **城乡分层**：独立模型测试流动过程在城乡间是否存在差异
- **性别异质性**：男性/女性子样本揭示了性别化的流动模式
- **缺失数据透明化**：代码通过注释详细记录了关于收入数据缺失的发现

### 地理分析
- **分级统计图**：展示职业成就的年度省级地图
- **统一色阶**：全局 6 分位数断点确保跨年度的可比性
- **分面布局**：所有 7 年的数据展示在单一的 2×4 矩阵中，共享图例
- **边界处理**：地图使用 GADM 4.1 省级边界文件

## 数据要求与复现说明

### 原始数据 (未包含)

由于以下原因，**未包含** CFPS 原始数据集：
- **文件体积**：每轮数据约 100–500 MB；7 轮数据超过 GitHub 限制
- **访问限制**：CFPS 数据受机构访问协议约束；用户必须自行在 CFPS 官网注册获取

### 复现步骤

1. **获取 CFPS 数据**：在 [CFPS 官网](https://www.isss.pku.edu.cn/cfps/index.htm) 注册并下载
2. **组织文件夹结构**：
   ```
   CFPS Project/
   ├── 0_raw/              (原始数据 + 地图文件)
   ├── 1_do/               (Stata 脚本)
   ├── 2_temp/             (临时中间文件)
   ├── 3_output/           (最终表格与图表)
   ├── 4_log/              (Stata 日志)
   └── 5_doc/              (文档)
   ```
3. **更新路径**：修改 `master_1_clean_data.do` 第 7 行：
   ```stata
   global PROJ "C:\Your\Local\Path\To\CFPS Project"
   ```
4. **运行流水线**：按照 `master_1_clean_data.do` → `master_2_panel_build.do` → `master_3_descriptive_and_maps.do` → `master_4_base_regression.do` → 稳健性测试 → `R Code for Poster Figure and Other/*.R` 的顺序执行

### 软件要求

**Stata:**
- 版本：16 或更高（需安装 `pca`, `winsor2`, `estout`, `coefplot` 等插件）
- 关键插件安装：`ssc install estout`, `ssc install coefplot`, `ssc install winsor2` 等

**R:**
- 版本：4.0 或更高
- 关键包：`tidyverse`, `ggplot2`, `circlize`, `scales`, `readr`
- 安装命令：`install.packages(c("tidyverse", "circlize", "scales"))`
