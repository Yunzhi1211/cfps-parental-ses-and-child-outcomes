How Do Parental Socioeconomic Resources Shape Children's Occupational Outcomes?

A Quantile-Based Analysis Using CFPS (2010–2022)

Abstract

This project examines how parental socioeconomic status (SES) shapes children's occupational outcomes using CFPS 2010–2022. It focuses on distributional inequality, regional and temporal heterogeneity, and the mediating roles of human capital and social capital.
